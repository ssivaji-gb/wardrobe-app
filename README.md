# Wardrobe Management Web Application

A web application to manage your wardrobe and avoid duplicate clothing purchases.

## Features

- User registration and login
- Add new dresses with details (type, color, category, occasion, season)
- Duplicate detection (prevents adding same type+color+category)
- View wardrobe as cards with filtering and search
- Dashboard with statistics
- Mobile-friendly responsive design
